## Integration Check

18/20 connections wired correctly. 2 minor issues:

### MISS-01 (Low): Win95 overlay content extraction
The overlay JS at `index.astro:145` uses `querySelector('.window-content') || querySelector('.window-body')`. For win95, the correct container is `.window-body`, but `.window-content` appears first in DOM. Content text is correct; container class is semantically mismatched.

**Fix:** Theme-aware selector: prefer `.window-body` when `data-theme="win95"`.

### MISS-02 (Medium): Missing SSG fallback on subpages
`about.astro` and `work.astro` have `<html lang="en">` without `data-theme="retro"`. The anti-flash script handles this quickly, but a brief unstyled flash is possible on first load.

**Fix:** Add `data-theme="retro"` to `<html>` in both files.
